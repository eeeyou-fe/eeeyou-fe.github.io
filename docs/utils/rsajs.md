# RSA 加密

## 使用方法

``` js
<script setup>
import rsa from '@eeeyou/utils/rsa';
rsa(str, key);
</script>
```

## Params


参数 | 说明 | 类型 | 默认值 | 必传
-- | -- | -- | -- | --
str | 要加密的字符串 | string | - | true
key | 公钥 | string | - | true
